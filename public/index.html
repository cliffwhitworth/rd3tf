RD3TF

ReactJS, D3, TensorFlowJS

This is a learning project to see how ReactJS, D3, and TensorFlowJS work together. The project uses TensorFlowJS to build a linear regression model that uses selected features such as horsepower and weight to predict the miles per gallon (mpg).

Getting things ready.
This may take a few seconds.

The dataset is courtesy of Stephen Grider's course on Machine Learning with Javascript. The TensorFlow model returns the coefficient of determination and predicts mpg based on user inputs for learning rate, iterations, and predictor variables. D3 is used to create scatterplots of each of the features compared to miles per gallon. Website inspired by Start Bootstrap
